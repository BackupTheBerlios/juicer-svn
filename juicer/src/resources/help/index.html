Juicer User Manual

The Main Window

  • 1 - Menu Bar: general and specific menus for the modules
  • 2 - General Tool Bar: tool bar wit general, module independent actions
  • 3 - Module Tool Bar: tool bar with actions of the selected module, changes if another module was selected
  • 4 - Tab Bar: module selection
  • 5 - Tab Widget: module content
  • 6 - Dock Widget: shows some extra information like the part list of a download; you can move, release, and hide it
  • 7 - Status Bar: general status information

Modules

Juicer consists of several modules whereas the data of each is shown in its very own tab in the main window. There are six different modules:
  • Downlaods - current downloads
  • Uplaods - current uploads
  • Search - searches for files and their results
  • Server - information about servers
  • Shares - shared directories and files
  • Incoming - management of downloaded files
Each of these modules will be described in the following.

Downloads

The Downloads module is obviously the most important part of Juicer and therefor provides the most amount of information.
