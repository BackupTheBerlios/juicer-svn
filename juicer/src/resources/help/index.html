The Main Window

  • 1 - Menu Bar: general and specific menus for the modules
  • 2 - General Tool Bar: tool bar wit general, module independent actions
    opens the options dialog where you can configure Juicer as well as the appleJuice core
    show information of the appleJuice network
    opens a link list file and adds all links in it
    adjusts the width of all columns of the currently visible tree view
    adds all links that are currently in the clipboard
    adds the link of the input field
  • 3 - Module Tool Bar: tool bar with actions of the selected module, changes if another module was selected
  • 4 - Tab Bar: module selection
    Juicer consists of several modules whereas the data of each is shown in its very own tab in the main window (5). There are six different modules:
    Downloads: all downloads
    Uploads: all uploads
    Search: searching for files
    Server: available servers
    Shares: shared folders and files
    Incoming: management of downloaded files
  • 5 - Tab Widget: module content
  • 6 - Dock Widget: shows some extra information like the part list of a download; you can move, release, and hide it
  • 7 - Status Bar: general status information
    since when the core is connected to a server
    the version of the core
    speed of all downloads
    speed of all uploads
    amount of data downloaded since the core is running
    amount of data uploaded since the core is running
    amount of credits available

    • Downloads

    The Downloads module is obviously the most important part of Juicer and therefor provides the most amount of information. Every download is shown by one line in the tree view that can be expanded by clikcing on the little plus sign or by double clicking the row in order to show the users where you are downloading from or may download from. They are sorted by status:
    1. "active": meaning you are currently getting data from it
    2. "queueing": you are in the queue of an user and you may download from it if you wait or increase your power download
    3. "other": you are not acutally connected to these users by different reasons
    You can manipulate downloads by selecting them and clicking on a button in the toolbar or in the context menu.
    pauses the selected download(s)
    starts the selected download(s)
    cancels the selected download(s)
    shows the part list of the selected download(s) in a separate window
    changes the file name on the selected download(s)
    changes the filename(s) of the selected download(s) by the content of the clipboard; this operation preserves file suffixes and adds "_1", "_2", ... in front of the file suffix if multiple downloads are selected
    opens the selected download(s) with the application defined in the configuration
    changes the target folder of the selected download(s)
    copies the ajfsp link(s) of the selected download(s) into the clipboard
    creates a link list of ajfsp links of the selected download(s) and writes them to a file
    Additionally there are operation that infect downloads or the download view in general:
    removes finished and canceld downloads from the list
    hides (NOT removes) puased downloads
    enables/disabled the dock widget with the part list
